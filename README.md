# 미니게임: 골스톱 (Go Or Stop)
### 🔍 게임 설명
- 유명 보드게임 Quoridor의 변형 게임입니다.
- 2인 플레이로 가볍게 즐길 수 있습니다.
### 📝 플레이 방법
1. 2명의 플레이어가 턴을 주고받으며 경쟁합니다.
2. 목표는 맵의 상단에 있는 색깔 블록으로 도달하는 것입니다.
3. 한 턴에 할 수 있는 행동의 종류는 단 2개입니다.  
   a. 상하좌우로 인접한 칸으로 총 3번 이동 가능합니다.  
   b. 벽을 최대 3개 인접하게 이어 지을 수 있습니다.
4. 단, 위 과정에서 상대가 절대 목적지에 도달하지 못하게 할 수는 없습니다.
